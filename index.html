Introdução

Este painel foi desenvolvido como um teste para avaliar o conhecimento do candidato em Front-end e APIs.

Apresentação do Projeto

Este é um dashboard com o intuito de mostrar de forma clara e objetiva, os números da pandemia causada pela COVID-19 (doença causada pelo coronavírus SARS-CoV-2) no Brasil e no mundo. Foi utilizado as tecnologias HTML,CSS,JS e VUE para requisitar os dados da API.
